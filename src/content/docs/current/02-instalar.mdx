## Via CDN

### Version Por Defecto

```html
<link
  rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/cactus_css@4.0.0/v4/cactus4.min.css"
/>
```

### Version Sin Temas Ni Fuentes

```html
<link
  rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/cactus_css@4.0.0/v4/cactus4nt.min.css"
/>
```

## Via administrador de paquetes

Usa el administrador de paquetes que te guste más:

- npm

```sh
 npm i cactus_css
```

- pnpm

```sh
 pnpm i cactus_css
```

- bun

```sh
 bun i cactus_css
```

### Importar

Simplemente en la parte superior de tu app en cualquier framework web usa cualquiera de las 2 versiones según tu caso:

#### Por Defecto (Con tema y fuente):

```js
import "cactus_css/v4/cactus4.min.css";
```

#### Sin tema (Sin tema ni fuente; util para personalizar)

```js
import "cactus_css/v4/cactus4nt.min.css";
import "../styles/themes.css"; // <- ubicación de tu archivo css con las clases root.
```
