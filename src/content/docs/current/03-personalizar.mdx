Si deseas personalizar los colores y la tipografía solo tienes que cambiarlos en un archivo css, busca el archivo `cactus4.css` :

```css
@import url("https://fonts.googleapis.com/css2?family=Montserrat:wght@400;600;800&family=Inter:wght@300..600&display=swap");

:root {
  --font1: "Montserrat", system-ui, sans-serif;
  --font2: "Inter", system-ui, sans-serif;
  --space: 1.5rem;
  --radius: 0.7rem;
  --text: #15160d;
  --background: radial-gradient(hsl(220, 40%, 94%), hsl(220, 14%, 94%));
  --shadow: 0 8px 32px 0 rgba(0, 0, 0, 0.205);
  --primary: #46eb3d;
  --secondary: #6bd5f5;
  --accent: #9bc200;
  --green: #45d25a;
  --red: #dc3243;
  --yellow: #f2c94c;
  --blue: #2495ca;
  --dark: #0e0e0e;
  --light: #fefefe;
  --contrast: var(--light);
}

:root[data-theme="dark"] {
  --text: #f1f2e9;
  --background: #212121;
  --primary: #7be576;
  --secondary: #0a7392;
  --accent: #d8ff3d;
  --green: #45d25a;
  --red: #dc3243;
  --yellow: #f2c94c;
  --blue: #2495ca;
  --dark: #2b2b2b;
  --light: #f1f2e9;
  --contrast: var(--dark);
}
```

al ser un archivo css, realmente es muy fácil cambiar los valores que no te agraden, como las sombras, efectos hover y demás.

:::tip
Una vez realices los cambios en el archivo css, si tu editor tiene un formateador recuerda comprimir el archivo css.
:::

Pagina para comprimir CSS:

<a href="https://www.toptal.com/developers/cssminifier">CSS Compressor</a>
<br />

También puedes usar una clase y modificarla, siendo más especifico o usando ´!important´

### Crear Tema

Si deseas crear un tema nuevo, solo tienes que crear un archivo css y definir una clase, poniendo las variables que deseas cambiar(las variables que admite son las que se muestran arriba en root) :

```css
:root[data-theme="dark"] {
  --text: #f1f2e9;
  --background: #212121;
  --primary: #7be576;
  --secondary: #0a7392;
  --accent: #d8ff3d;
  --green: #45d25a;
  --red: #dc3243;
  --yellow: #f2c94c;
  --blue: #2495ca;
  --dark: #2b2b2b;
  --light: #f1f2e9;
  --contrast: var(--dark);
}
```

:::tip
Puedes usar este increíble recurso para crear los temas https://www.realtimecolors.com/ . Al darle export usa css y activa el checkbox de Themes, para tener el tema dark.
:::

### Cambio de tema usando JS

Para cambiar el tema solo tienes que cambiar la clase del body o el html, por ejemplo:

```html
/* Definimos un checkbox */
<div class="toggle-btn" id="light-mode-toggle">
  <label title="Use light theme" aria-label="Use light theme">
    <input type="checkbox" name="light-mode-toggle" />
    <span></span>
  </label>
</div>
/*Las clases para crear el checkbox son de un css aparte */
```

Ahora en el script

```js
// creamos una constante para el checkbox
const toggle = document.getElementById("light-mode-toggle");
// creamos un evento para el checkbox
toggle.addEventListener("change", (e) => {
  // si el checkbox esta desactivado
  if (e.target.checked == false) {
    // guardamos en el localStorage que el modo oscuro esta desactivado
    localStorage.setItem("dark", "false");
    // removemos el atributo data-theme
    document.documentElement.removeAttribute("data-theme");
  } else {
    // si el checkbox esta activado guardamos en el localStorage que el modo oscuro esta activado
    localStorage.setItem("dark", "true");
    // agregamos el atributo data-theme con el valor "dark"
    document.documentElement.setAttribute("data-theme", "dark");
  }
});

// comprobamos si el modo oscuro esta activado
if (localStorage.getItem("dark") === "true") {
  // si esta activado activamos el checkbox
  toggle.querySelector("input").checked = true;
  // agregamos el atributo data-theme con el valor "dark"
  document.documentElement.setAttribute("data-theme", "dark");
}
```

:::note
Si deseas cambiar el tema con un botón, solo tienes que cambiar el input por un button y agregarle un evento onclick.
Nota : si estas usando Astro Js, usa

```js
<script is:inline>...</script>
```

:::
